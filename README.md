# SMS Spam Classifier

## Problem
Build a machine learning model to classify SMS messages as spam or ham.

## Dataset
UCI SMS Spam Dataset

## Technologies
Python
Scikit-learn
NLP
Streamlit
MySQL

## Model
TF-IDF + SVC

## Performance
Accuracy: 98%
Precision: 97%
F1-score: 94%

## Run Project

pip install -r requirements.txt
streamlit run app.py
